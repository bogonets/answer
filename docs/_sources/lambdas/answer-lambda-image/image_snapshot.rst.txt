image_snapshot
=============================

이미지 스탭샷을 저장하고 출력으로 보낸다.

입력 슬롯
---------

* **list**

출력 슬롯
---------

* **list**

속성 목록
---------

* **스냅샷 플래그**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| read_and_write  + bool  + True     + False         |
+-----------------+-------+----------+---------------+

스냅샷 플래그를 활성화 한다. 람다에 신호가 전파되면 이미지가 저장된다.
