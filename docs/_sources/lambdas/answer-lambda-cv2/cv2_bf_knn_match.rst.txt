cv2_bf_knn_match
=============================

쿼리 집합에서 각 설명자에 대해 가장 일치하는 k 개의 항목을 찾습니다.

입력 슬롯
---------

* **query**

* **train**

출력 슬롯
---------

* **match1**

* **match2**

속성 목록
---------

* **normType**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| initialize_only + str   + True     + L2            |
+-----------------+-------+----------+---------------+

NORM 의 종류

* **crossCheck**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| initialize_only + bool  + True     + False         |
+-----------------+-------+----------+---------------+

false 인 경우 각 쿼리 설명자에 대해 k 개의 가장 가까운 이웃을 찾을 때 기본 BFMatcher 동작이됩니다.

* **k**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| read_and_write  + int   + True     + 2             |
+-----------------+-------+----------+---------------+

각 쿼리 설명자당 발견 된 최상의 일치 수.
