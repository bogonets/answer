minio_put_object2
=============================

오브젝트 스토리지 서버에 새로운 오브젝트를 추가한다.

입력 슬롯
---------

* **bucket_name**

* **object_name**

* **data**

속성 목록
---------

* **URL**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| initialize_only + str   + True     +               |
+-----------------+-------+----------+---------------+

오브젝트 스토리지 서버의 주소.

* **Access KEY**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| initialize_only + str   + True     +               |
+-----------------+-------+----------+---------------+

액세스 키는 계정을 고유하게 식별하는 사용자 ID와 같습니다.

* **Secret KEY**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| initialize_only + str   + True     +               |
+-----------------+-------+----------+---------------+

비밀 키는 계정의 비밀번호입니다.

* **Secure**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| initialize_only + bool  + True     + false         |
+-----------------+-------+----------+---------------+

'True'로 선택할 경우 HTTPS 보안 접속을 적용한다.

* **확장자**

+-----------------+-------+----------+---------------+
| Rule            + Type  + Required + Default value |
+=================+=======+==========+===============+
| initialize_only + str   + True     + .txt          |
+-----------------+-------+----------+---------------+

확장자 타입을 설정한다.
