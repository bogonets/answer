# 관리자 설정

## 개요

서버 정보 및 통계 정보를 출력합니다.

![admin-overview](./images/admin-overview.png)

## 사용자

가입된 전체 사용자 목록을 출력합니다.

![admin-users](./images/admin-users.png)

### 새로운 사용자

사용자 목록 테이블 우측 상단에 위치한 `새로운 사용자` 버튼을 클릭하면 사용자를 추가할 수 있습니다.

![admin-users-new](./images/admin-users-new.png)

**(필수)** 로 지정된 항목은 반드시 입력해야 합니다. 또한 **관리자** 스위치로 해당 권한을 부여할 수 있습니다.

### 사용자 편집

기본 사용자 정보를 수정할 수 있습니다.

![admin-users-edit](./images/admin-users-edit.png)

**관리자** 스위치로 해당 권한을 제거 또는 설정할 수 있습니다.

### 그룹 편집

그룹의 기본 정보를 수정할 수 있습니다.

![admin-groups-edit](./images/admin-groups-edit.png)

## 권한

멤버 관리에 필요한 권한 규칙을 확인할 수 있습니다.

![admin-permissions](./images/admin-permissions.png)

제공되는 기본 권한 목록은 다음과 같습니다:

- 레이아웃 보기 - `레이아웃` 페이지를 볼 수 있습니다.
- 레이아웃 편집 - `레이아웃` 페이지의 기능을 수정할 수 있습니다.
- 파일 보기 - `파일` 페이지를 볼 수 있습니다.
- 파일 편집 - `파일` 페이지의 **파일(FILE)** 을 생성/수정/제거 할 수 있습니다.
- 테이블 보기 - `테이블` 페이지를 볼 수 있습니다.
- 테이블 편집 - `테이블` 페이지의 스키마 편집 및 열(ROW) 생성/수정/제거 할 수 있습니다.
- 태스크 보기 - `태스크` 페이지를 볼 수 있습니다.
- 태스크 편집 - `태스크` 페이지의 작업(Process)을 제어할 수 있습니다.
- 비주얼 프로그래밍 보기 - `비주얼 프로그래밍` 페이지를 볼 수 있습니다.
- 비주얼 프로그래밍 편집 - `비주얼 프로그래밍` 페이지의 람다를 수정하거나 재배치할 수 있습니다.
- 회원 보기 - 그룹 또는 프로젝트의 회원 목록을 볼 수 있습니다.
- 회원 편집 - 그룹 또는 프로젝트의 회원 목록을 편집할 수 있습니다.
- 설정 보기 - 그룹 또는 프로젝트의 설정 정보를 볼 수 있습니다.
- 설정 편집 - 그룹 또는 프로젝트의 설정 정보를 편집할 수 있습니다.
- 도메인 제거 - 그룹 또는 프로젝트를 제거할 수 있습니다.

그 밖의 권한은 플러그인 스크립트에서 직접 추가해야 한다.

### 새로운 권한

권한 목록 테이블 우측 상단에 위치한 `새로운 권한` 버튼을 클릭하면 권한을 추가할 수 있습니다.

![admin-permissions-new](./images/admin-permissions-new.png)

- `권한` : 메뉴나 기능에 해당하는 전체 권한을 추가하거나 제거할 수 있습니다.
- `숨김` : 일반 사용자의 멤버 관리 화면에 노출 여부를 결정할 수 있습니다.
- `잠금` : 수정 및 삭제가 불가능 하도록 제한합니다.

### 권한 편집

권한의 기본 정보를 수정할 수 있습니다.

![admin-permissions-edit](./images/admin-permissions-edit.png)

## 구성

관리자 레벨의 엔서 설정 정보를 확인할 수 있습니다.

![admin-configs](./images/admin-configs.png)

- 공개 가입 활성화 : 활성화하면 방문자가 직접 가입할 수 있습니다.
- 액세스(Access) 토큰 기간 : 보안을 위해 토큰을 지속적으로 갱신합니다.
- 갱신(Refresh) 토큰 기간 : 보안을 위해 강제 로그아웃 합니다.
- 외부 호스트 : **DEPRECATED**
- HTTP 시간 초과 : **DEPRECATED**
- 사용자 : **DEPRECATED**
- 그룹 : **DEPRECATED**
- 로그 레벨 : 시스템 관리자를 위한 디버깅 옵션 입니다.
- 상세함(Verbose)정도 : 시스템 관리자를 위한 디버깅 옵션 입니다.
