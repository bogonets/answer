시작하기#

로그인#

엔서 페이지에 접속하면 다음과 같은 화면이 출력됩니다.

계정명 과 비밀번호를 입력 후 로그인 버튼을 클릭하면 메인 페이지로 이동합니다.

페이지 우측 상단의 메뉴는 각각

  • : 밝은/어두운 테마 변경

  • : 언어 변경

  • : API 서버 주소 변경 (이 버튼은 Production 배포시 노출되지 않습니다)

회원 가입#

만약 가입한 계정이 없다면 하단의 회원 가입 링크를 클릭하여 계정을 생성하면 됩니다.

(필수) 로 지정된 항목은 반드시 입력해야 합니다.

메인 페이지#

로그인에 성공하면 첫 번째로 이동하는 페이지 입니다.

이 곳 에서는 소속된 그룹 목록과 프로젝트 목록이 출력됩니다.

페이지 좌측 최 상단에 있는 ANSWER 로고를 클릭하면 현재 보고 있는 페이지로 이동할 수 있습니다.

  • GROUP 버튼을 클릭하면 그룹 목록 페이지로 이동,

  • PROJECT 버튼을 클릭하면 프로젝트 목록 페이지로 이동,

사용자 메뉴#

페이지 우측 최 상단에 있는 사용자 아이콘()을 클릭하면 사용자 메뉴가 출력됩니다.

출력되는 메뉴는 각각

  • : 개인 설정 페이지로 이동합니다.

  • : 관리자 설정 페이지로 이동합니다. (이 목록은 관리자 계정만 노출됩니다)

  • : 엔서에 대해 페이지로 이동합니다.

  • : 계정을 로그아웃 하고 로그인 페이지로 이동합니다.
