# @recc/api

Frontend API for the recc core

## Project setup
```.shell
yarn install
```

### Compiles and minifies for production
```.shell
yarn build
```

### Style check
```.shell
yarn style
```

### Lints and fixes files
```.shell
yarn lint
```

### Run your unit tests
```.shell
yarn test
```
