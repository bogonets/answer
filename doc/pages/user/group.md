# 그룹

그룹은 `프로젝트`의 집합 입니다.

## 그룹 목록

그룹 목록은 소속된 전체 그룹 목록을 볼 수 있으며,
필요하다면 `새로운 그룹` 버튼을 클릭하여 새로운 그룹을 생성할 수 있습니다.

![groups](./images/groups.png)

하나의 그룹을 선택하면 해당 그룹 페이지로 이동합니다.

## 프로젝트

현재 그룹에서 생성된 프로젝트 목록을 확인할 수 있습니다.

![group-projects](./images/group-projects.png)

### 새로운 프로젝트

프로젝트 목록 테이블 우측 상단에 위치한 `새로운 프로젝트` 버튼을 클릭하면 프로젝트를 추가할 수 있습니다.

![group-projects-new](./images/group-projects-new.png)

## 회원 관리

현재 그룹에 가입된 멤버와 각 멤버의 권한을 확인할 수 있습니다.

![group-members](./images/group-members.png)

## 그룹 설정

그룹의 기본 속성을 확인하거나 수정할 수 있습니다.

![group-settings](./images/group-settings.png)
